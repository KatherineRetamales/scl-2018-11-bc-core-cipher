Bienvenido(a) a SecretCode!
SecretCode

¿Qué es y para qué sirve?

Es una aplicación web que sirve para poder cifrar o descifrar los mensajes secretos entre agentes de seguridad o protegidos.

¿Cómo se ocupa?

1 - En el recuadro superior se ingresa el texto que se quiere cifrar o descifrar.

2 - Se ingresa el número de desplazamiento.

3 - Luego se pincha el botón cifrar o descifrar y en el recuadro de más abajo aparecerá el texto descifrado o cifrado según corresponda.

4 - Se puede borrar y copiar el texto.
